Diccionario de Domótica

  • Automatización

    Proceso de hacer que un sistema o aparato funcione automáticamente.

  • Internet de las Cosas (IoT)

    Red de dispositivos interconectados que pueden comunicarse entre sí y con usuarios.

  • Sensor

    Dispositivo que detecta cambios en el entorno y envía esa información a otros dispositivos.

  • Actuador

    Componente que recibe una señal y realiza una acción física.

  • Controlador

    Dispositivo que dirige el funcionamiento de otros dispositivos en un sistema domótico.

  • Hogar Inteligente

    Casa equipada con dispositivos conectados que pueden ser controlados de manera remota.

  • Asistente de Voz

    Dispositivo o software que responde a comandos de voz y puede controlar otros dispositivos.

  • Domótica

    Conjunto de tecnologías aplicadas al control y automatización inteligente del hogar.

  • Protocolo de Comunicación

    Conjunto de reglas que permiten la comunicación entre dispositivos.

  • Conectividad

    Capacidad de los dispositivos para conectarse y comunicarse entre sí.

  • Inteligencia Artificial

    Tecnología que permite a los dispositivos aprender y tomar decisiones basadas en datos.

  • Red Inalámbrica

    Sistema de comunicación que no requiere cables físicos para la transmisión de datos.

  • Automatización de Iluminación

    Sistema que permite controlar las luces de una casa de manera automática o remota.

  • Seguridad Domótica

    Conjunto de dispositivos y sistemas que mejoran la seguridad del hogar.

  • Monitorización Remota

    Capacidad de supervisar y controlar dispositivos a distancia.

  • Interruptor Inteligente

    Dispositivo que permite controlar el encendido y apagado de aparatos eléctricos de forma remota.

  • Cerradura Inteligente

    Dispositivo de cierre que puede ser controlado mediante una aplicación móvil o sistema remoto.

  • Termostato Inteligente

    Dispositivo que regula la temperatura de una casa de forma automática o remota.

  • Control por Voz

    Funcionalidad que permite operar dispositivos mediante comandos verbales.

  • Automatización de Cortinas

    Sistema que permite abrir y cerrar cortinas de manera automática o remota.
